# 目录结构

服务器 `/root/env/` 目录组织。

```
/root/env/
├── docker/              # Docker 容器服务
│   ├── clash-conf/      # Clash 代理配置
│   ├── frps/            # FRP 内网穿透配置
│   ├── mysql-conf/      # MySQL 配置
│   ├── docker-compose.yml
│   └── README.md
├── log/                 # 日志目录
│   └── iops_monitor/    # IOPS 监控日志
├── iops_monitor.sh      # 磁盘 IOPS 监控脚本
└── .claude/             # Claude AI 配置
```
