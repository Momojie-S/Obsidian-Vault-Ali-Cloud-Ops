# 端口分配

> 服务器端口占用总览，新服务部署前先查阅避免冲突。

## 端口分配表

| 端口 | 服务 | 说明 |
|------|------|------|
| 7890 | Clash | 代理服务（Docker host 模式） |
| 18789 | OpenClaw | WeCom 回调监听 |
| 21001 | MySQL | 数据库服务（Docker） |
| 21002 | OneDragon-Alpha 前端 | Vue 开发服务器 |
| 21003 | OneDragon-Alpha 后端 | FastAPI/Uvicorn |
| 21004 | OneDragon-Alpha Chrome 调试 | chrome-devtools-mcp 远程调试 |
| 21101 | momojie-home Chrome 调试 | chrome-devtools-mcp 远程调试 |
| 21102 | momojie-home 开发服务器 | VuePress 博客开发服务器 |
| 22001 | FRPS | frp 服务端口 |
| 22002 | FRPS | SSH 映射端口 |

## 端口规划

- **21xxx** - 开发项目相关
  - 210xx - OneDragon-Alpha
  - 211xx - momojie-home
- **22xxx** - FRP 内网穿透
- **18789** - OpenClaw 专用
- **7890** - Clash 代理（固定）

## 查看当前端口占用

```bash
# 查看所有监听端口
ss -tlnp

# 查看特定端口
ss -tlnp | grep <端口>
```
