# Docker 镜像加速器

## 阿里云镜像加速器

配置阿里云镜像加速器，提升国内拉取 Docker Hub 镜像的速度。

### 获取加速器地址

1. 登录 [阿里云容器镜像服务](https://cr.console.aliyun.com/cn-hangzhou/instances/mirrors)
2. 左侧菜单：镜像工具 → 镜像加速器
3. 复制专属加速器地址（格式：`https://<你的ID>.mirror.aliyuncs.com`）

### 配置方法

**方法一：修改 daemon.json**

```bash
sudo mkdir -p /etc/docker
sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["https://<你的ID>.mirror.aliyuncs.com"]
}
EOF

# 重载配置
sudo systemctl daemon-reload
sudo systemctl restart docker
```

**方法二：通过 systemd drop-in**

```bash
sudo mkdir -p /etc/systemd/system/docker.service.d
sudo tee /etc/systemd/system/docker.service.d/mirror.conf <<-'EOF'
[Service]
ExecStart=
ExecStart=/usr/bin/dockerd --registry-mirror=https://<你的ID>.mirror.aliyuncs.com
EOF

sudo systemctl daemon-reload
sudo systemctl restart docker
```

### 验证配置

```bash
docker info | grep -A 5 "Registry Mirrors"
```

### 注意事项

- 每个 Alibaba Cloud 账号有专属的加速器地址
- 加速器仅对 Docker Hub 公共镜像生效
- 私有仓库需单独配置认证
